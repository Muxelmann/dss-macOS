# DSS for macOS `Swift`

## Summary

This project relies on [`electricdss-src`](https://github.com/PMeira/electricdss-src) and [`dss_capi`](https://github.com/PMeira/dss_capi) to build a DSS module that is ready to be used in a `Swift` based project.

## Build

```bash
make setup
make
make install
```
